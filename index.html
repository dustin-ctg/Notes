Bobby Baldor Postcard Project

Project Overview

I want to get a message to every owner of record (this is a short-term rental community so needs to go to address of record not property). And maybe, one asap but also another follow up in a few weeks later. Want to get the attention of every owner that this is a hot market and units in this specific community are in very, very high demand.
Link here: 3077 Pequod Place

  • Need rental property owner list
    • REiSource
      • Usually pretty accurate
      • Can filter by a lot of different factors
    • Corefact lists
      • Can order directly from postcard design screen
      • Usually pretty accurate
      • Limited filtering factors, but might work fine for this
  • Might need to have a campaign set up
    • Corefact Elite campaign is a 12-month program
    • Just sold/listed one-off orders are pretty easy to set up
  • Getting a Design
  • From Fiverr:
    1. Go to Fiverr.com
    2. Create an Account
    3. Go to Home
    4. Hover over Graphics & Design
    5. Go to Print Design > Postcard Design
    6. Select a designer that you would like to work with, usually you want one that specifies that they do EDDM postcards
    7. Make sure to give your designer the print specifications
    8. If you want to use the design template I made, I've included that in the package, otherwise, they can design something custom for you as well.
  • On Corefact (for custom designs):
    1. Go to Corefact.com
    2. Sign up or login
    3. Go to Products
    4. Click "Upload Your Design" in the left sidebar, then "Upload – Postcards"
    5. Click "Jumbo" under sizes, then "Upload – Jumbo"
    6. Click "Continue", name the order, and click "Continue" again
    7. Upload the postcard PDF that the designer sent you, they should have set it up with all the necessary specifications. Click "Continue"
    8. Import or pick your list. You have a few options for this:
      • If you got your list from REiSource:
        • Click "Import List" and select the .csv file you downloaded from REiSource
        • Map all necessary information according to how it's displayed in the .csv. You'll need:
          • Name of addressee. Corefact will guide you through how the name needs to be set up, but it should be: Contact Info > Name > Last Name (for last name field), and First Name for the other
          • Street address. The map location is at: Site Address > Site Street Address > Site House Number and Street Name
          • City-State-Zip. Sometimes this is all in one field and sometimes it's in separate fields. The basic mapping route is the same: Site Address > (select the field that applies)
        • Click "Continue" and you'll be asked if you want to save the list for later, which is useful for future mailings.
      • Ordering Corefact Lists:
        • Can order directly from postcard design screen
        • Usually pretty accurate
        • Limited filtering factors, but might work fine for this
